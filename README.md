# e-plantShopping

## Project Overview
**e-plantShopping** is a simple React-based shopping application designed for plant enthusiasts.  
It allows users to browse different plants, add them to a shopping cart, and manage their purchases in an intuitive interface.

### Features
- 🪴 Browse a catalog of plants
- 🛒 Add/remove items from the shopping cart
- 📦 Track your selected items before checkout
- 🎨 Clean and responsive UI built with React

---

This project demonstrates frontend development skills with React, focusing on component-based architecture, state management, and user-friendly design.
